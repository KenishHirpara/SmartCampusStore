# 🏫 SmartCampusStore

A simple Student Campus Store Web Application where users can browse products and place orders easily.

---

## 🚀 Features

- User Registration & Login  
- View Products  
- Add to Cart  
- Place Orders  
- Admin Product Management  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB  

---

## ⚙️ How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/KenishHirpara/SmartCampusStore.git
   ```

3. Install dependencies
   ```bash
   cd server
   npm install
   ```

5. Start the server
   ```bash
   npm start
   ```
  
7. Open frontend  
Open `index.html` in your browser
